## Create Video

**Host URL:** `https://qr-be-api-vvafsac2oa-uc.a.run.app`

**Endpoint:** `/subtitle`

**Method:** `POST`

### Headers

```markdown
Content-Type: application/json
x-api-key: <your_api_key> 
```

### Request ( Body )



Key              | Data Type | Description                                                                       | Mandatory
-----------------|-----------|-----------------------------------------------------------------------------------|-----------
videoUrl         | STRING    | URL of the source video (YouTube link or video file URL)                            | Required
callBackUrl      | STRING    | URL to receive the response after all tasks are completed                          | Required
subtitleTemplate | STRING    | Choose a subtitle template (default is Impact). Pass as "false" to skip.           | Optional
videoTitle       | STRING    | Title of your video                                                               | Optional
backgroundMusic  | BOOLEAN   | Add AI-generated music matching the mood of your video. Set to true to enable       | Optional
aiBRoll          | BOOLEAN   | Add AI-generated B-roll footage to the video. Set to true to enable                 | Optional
