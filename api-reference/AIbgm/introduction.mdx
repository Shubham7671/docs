## How to Create a Video Clip

**Base URL:** `https://mango.quickreel.io/api/v2`

**Endpoint:** `/bgm`

**Method:** `POST`

**Headers**

```markdown
Content-Type: application/json
x-api-key: <your_api_key>
```

**Request Body Parameters**

| Key        | Data Type | Description                                              | Mandatory |
| ---------- | --------- | -------------------------------------------------------- | --------- |
| videoUrl   | STRING    | URL of the source video (YouTube link or video file URL) | Required  |
| webhookUrl | STRING    | URL to receive processing updates                        | Required  |

### Advanced Use Cases

The AI-powered background music (BGM) feature analyzes the emotional tone and context of your video content to automatically select and integrate suitable background music. This enhances viewer engagement and enriches the overall experience, making it ideal for:

- **Marketing Videos:** Elevate your brand message with music that resonates with your audience.
- **Entertainment Content:** Add depth and emotion to your storytelling.
- **Educational Materials:** Maintain viewer interest and emphasize key points with appropriate audio cues.

By leveraging AI emotion detection, the BGM tool ensures that the music complements the video's mood, creating a seamless and immersive experience.
