## Create Video

**Host URL:** `https://qr-be-api-vvafsac2oa-uc.a.run.app`

**Endpoint:** `/api/request/bgm`

**Method:** `POST`

### Headers

```markdown
Content-Type: application/json
x-api-key: <your_api_key> 
```

### Request ( Body )



Key              | Data Type | Description                                                                       | Mandatory
-----------------|-----------|-----------------------------------------------------------------------------------|-----------
videoUrl         | STRING    | URL of the source video (YouTube link or video file URL)                            | Required
callBackUrl      | STRING    | URL to receive the response after all tasks are completed                          | Required
silenceRemover   | BOOLEAN   | Remove silences in the video. Set to true to enable                                 | Optional
noiseCleaner     | BOOLEAN   | Enhance the audio quality. Set to true to enable                                    | Optional
