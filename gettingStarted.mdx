### 1. Request Your API Key

<Note>
  To use the QuickReel API, you'll need an API key. Get yours from the
  [QuickReel Dashboard](https://api.quickreel.io/).
</Note>

### 2. Base URL

All API requests should be made to:

```bash
https://engine.quickreel.io/api/v2
```

### 3. Authentication

<CodeGroup>
```bash cURL
curl --request POST \
  --url https://engine.quickreel.io/api/v2/clip \
  --header 'x-api-key: YOUR_API_KEY' \
  --header 'Content-Type: application/json'
```

```javascript Node.js
const headers = {
  "x-api-key": "YOUR_API_KEY",
  "Content-Type": "application/json",
};
```

```python Python
headers = {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}
```

</CodeGroup>

## API Response Format

All API responses follow a consistent JSON format:

```json
{
  "statusCode": 200,
  "status": "success",
  "message": "Operation completed successfully",
  "data": {
    // Response data specific to the endpoint
  }
}
```
